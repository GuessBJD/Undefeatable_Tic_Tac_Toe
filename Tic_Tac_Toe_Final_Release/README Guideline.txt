# CodeBlock TicTacToe
An AI that acts as the opponent to play TicTacToe. Using CodeBlocks, the open source, cross platform IDE

# Prerequisites before running the code:
-You are required to download the Code::Block 
-The current release we used to develope the TicTacToe game is:
 Release 20.03 rev 11983 (2020-03-12 18:24:30) qcc 8.1.0 Windows/unicode - 64 bit
 From the link-> https://www.fosshub.com/Code-Blocks.html?dwl=codeblocks-20.03-setup.exe
 
# Description about this code folder:
- you may have a look on the explanation of coding and neat configuration before compile and run the program.'
As you can see the Final Project file contains three modules for different usage. 
->Tic_Tac_Toe_Final_v1.2_Analysis_Data_Sets_Simulation is used for the report which analyse the sense of rationality of the AI.
->Tic_Tac_Toe_Final_v1.2_Bot_v_Bot_Mod is used to show how AI make moves rationally
->Tic_Tac_Toe_Final_v1.2 creates an interface for the HUMAN player to have an experience of playing the TicTacToe match with the Artificial Intelligence.

# Description on how to run the code:
1. After installing the Code::Block with the version given above, you are required to open the project.
   For Viewing purposes, open -> Tic_Tac_Toe_Final_v1.2.1_Bot_v_Bot_Mod.cbp
   For Analysing purposes, open -> Tic_Tac_Toe_Final_v1.2.1_Analysis_Data_Sets_Simulation.cbp
   For Playing purposes, open -> Tic_Tac_Toe_Final_v1.2.1
2. Compile and Run the code.

!!! The projects in the file only support by the Code::Block version given above, other version of Code::Block may or may not be compiled sucessfully.
    Since this is the version we used during the development of the TicTacToe game.
